<h1 align="center">Puerto Bot</h1>

<p align="center">
<strong>Bot para servidor de RPG de futebol no Discord</strong>
</p>

<p align="center">
Puerto é um bot desenvolvido para automatizar e auxiliar o funcionamento de um servidor
de <strong>RPG de futebol</strong> no Discord, permitindo que jogadores participem de ligas,
interajam com o sistema e organizem partidas dentro da comunidade.
</p>

<hr>

<h2>Sobre o Projeto</h2>

<p>
O <strong>Puerto</strong> foi criado para um servidor de Discord focado em
<strong>RPG de futebol</strong>, onde os membros podiam assumir diferentes papéis dentro
do universo do futebol, como:
</p>

<ul>
<li>Jogadores</li>
<li>Técnicos</li>
<li>Jornalistas</li>
<li>Empresários</li>
<li>Clubes e ligas</li>
</ul>

<p>
O bot ajudava a organizar as interações do servidor, automatizando processos
e oferecendo comandos para facilitar a experiência dos usuários.
</p>

<hr>

<h2>🚀 Funcionalidades</h2>

<ul>
<li>Comandos relacionados ao RPG de futebol</li>
<li>Organização de ligas e partidas</li>
<li>Sistema de interação entre usuários</li>
<li>Automação de respostas e comandos</li>
<li>Ferramentas administrativas para o servidor</li>
</ul>

<hr>

<h2>Tecnologias Utilizadas</h2>

<ul>
<li><strong>Python</strong></li>
<li><strong>discord.py</strong></li>
<li><strong>APIs do Discord</strong></li>
</ul>

<p>
A biblioteca <strong>discord.py</strong> permite criar bots capazes de interagir com
servidores, responder comandos e automatizar tarefas dentro do Discord.
</p>

<hr>

<h2>⚙️ Como executar o bot</h2>

<h3>1️⃣ Clonar o repositório</h3>

<pre>
git clone https://github.com/AbnerSantosAlves/botPuerto.git
</pre>

<h3>2️⃣ Entrar na pasta do projeto</h3>

<pre>
cd botPuerto
</pre>

<h3>3️⃣ Instalar dependências</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>4️⃣ Configurar o token do bot</h3>

<p>
Crie um arquivo de configuração ou variável de ambiente com o token do bot:
</p>

<pre>
DISCORD_TOKEN=seu_token_aqui
</pre>

<h3>5️⃣ Executar o bot</h3>

<pre>
python main.py
</pre>
